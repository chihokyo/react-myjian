# React项目笔记

## 项目搭建

### 新建项目

```
npx create-react-app react-myjianshu
cd react-myjianshu
```

### Github关联

```
# 关联库
git remote add origin git@github.com:chihokyo/react-myjian.git
git branch -M master
git push -u origin master
# 创建develop分支并推送
git checkout -b develop
git push origin develop
```
